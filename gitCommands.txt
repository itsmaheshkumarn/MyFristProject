git status                >> It will show the current status.
git add file.txt          >> It will move the file from Working Dir to Staging Dir.
git commit -m "comment"   >> It will do commit and move the file to Local Repository.
git commit -a -m "commit" >> It will skip the stagging part and move the file from Working Dir to Local Repo.
git log                   >> It will show the info about the commit file.
git diff                  >> It will show the newly added data in Working Dir.
git diff --staged         >> It will show the newly added data in Staging Dir.
git rm --cached file.txt  >> It will delete the file from Local Repo and keep it in Working Dir(Untracked).
